FANTASY PREMIER LEAGUE DATA SOURCES SUMMARY
============================================

1. FPL-Elo-Insights (Primary Recommendation)
   Source: https://github.com/olbauday/FPL-Elo-Insights
   
   STRENGTHS:
   - Active maintenance with twice-daily updates (5:00 AM and 5:00 PM UTC)
   - Enhanced coverage including pre-season friendlies, domestic cups, European competitions
   - Advanced defensive metrics (Clearances, Blocks, Interceptions, Tackles)
   - Multiple data formats organized by gameweek, tournament, season-level
   - Direct API access via raw GitHub URLs
   
   DOWNSIDES:
   - Limited historical coverage (mainly current season)
   - Dependent on single maintainer
   
   AVAILABLE DATA:
   - Current season gameweek data (merged_gw.csv)
   - Season summary data (season_summary.csv)
   - Player statistics with enhanced metrics
   - Cup competitions data
   - Elo ratings and advanced analytics

2. Vaastav's Repository (Historical Data)
   Source: https://github.com/vaastav/Fantasy-Premier-League
   
   STRENGTHS:
   - Extensive historical coverage from 2016-2024
   - Well-documented data structure with cleaned CSV files
   - Community trust (1,600+ stars, widely cited)
   - Direct Python access using pandas
   - Complete season data for multiple years
   
   DOWNSIDES:
   - Weekly updates stopped after 2024-25 season
   - Only 3 major updates planned per season going forward
   - Limited current season updates
   
   AVAILABLE DATA:
   - Historical player data by gameweek (players_raw.csv)
   - Individual player histories (player_idlist.csv)
   - Team data (teams.csv)
   - Fixtures data
   - Bootstrap static data
   - Cleaned player data (cleaned_players.csv)

3. Official FPL API
   Source: https://fantasy.premierleague.com/api/
   
   STRENGTHS:
   - Official source with guaranteed accuracy
   - Real-time updates
   - Live data for current season
   - Comprehensive current season data
   
   DOWNSIDES:
   - No historical data access
   - Rate limiting restrictions
   - Current season only
   - No enhanced metrics
   
   AVAILABLE DATA:
   - Bootstrap static data (teams, players, events, settings)
   - Live fixtures data
   - Manager team data
   - Player statistics (basic)
   - Gameweek information
   - Transfer deadlines

RECOMMENDED WORKFLOW:
====================
- Current Season: Use FPL-Elo-Insights for real-time updates and enhanced metrics
- Historical Analysis: Use Vaastav's repository for seasons 2016-2024
- Live Data: Use Official FPL API for real-time current season data
- Combine Sources: Merge datasets for comprehensive multi-season analysis
